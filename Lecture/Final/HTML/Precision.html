1. Floating Point with 8 Bytes

1.1. Representation : E4M3

  • Normal : Non-zero E
    • (−1)s×(1.0+0.M)×2E−bias(-1)^s \times (1.0 + 0.M) \times 2^{E-bias}(−1)s×(1.0+0.M)×2E−bias
    • bias = 7
  • Subnormal : Zero E
    • (−1)s×(0.0+0.M)×2−6(-1)^s \times (0.0 + 0.M) \times 2^{-6}(−1)s×(0.0+0.M)×2−6

1.2. QAT(Quantization-Aware Training)

  1. Train network in full precision(fp16)
  2. Quantize FP weight and execute model
  3. Back-propagate the error (using quantized weights/activations and full-precision gradient) and update the FP weights

1.3. Quantization

  1. Calculating Scale(S-value) and Zero(Z-value)
  • S=(MAX−MIN)/255S = (MAX-MIN)/255S=(MAX−MIN)/255
  • Z=round(MIN/S)Z = round(MIN/S)Z=round(MIN/S)
  1. Original to Quantization(Original real)
  • q=round(x/S)+Zq = round(x/S) + Zq=round(x/S)+Z
  1. Quantization to Original(Fake real)
  • R′=S(q−Z)R' = S(q - Z)R′=S(q−Z)

1.4. STE(Stright Through Estimator)

  • Quantization is not differentiable
    • STE solve this problem by estimator function
    • identifty function : function that well approximates the quantization function and is differentiable
    • so, differentiating the estimator function rather than the quantization function
